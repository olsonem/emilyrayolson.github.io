Browser Game: Set your House on Fire

Project Description

This project is designed to practice event handling in the context of a modern JavaScript web application. in this project, I will built a modified version of the classic Tic Tac Toe game called Set Your House on fire.

This project; creates a Player class that accepts an icon parameter in the constructor(), completes the creation of a TicTacToe class to handle the primary game functions, sets up the game board using DOM manipulation commands, initializes values to count moves and track game status (playing, won, draw), adds event listeners to game tiles to handle player interaction, creates methods to handle game logic including recording moves, switching players, etc, allows the player to begin a game by clicking the "Start Game" button, and adds event listeners to the document object to handle "win" and "draw" events.
